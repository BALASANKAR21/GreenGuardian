# **App Name**: GreenGuardian SmartPlant

## Core Features:

- User Onboarding & Location Detection: Welcome users and automatically detect their city using the IPinfo API.
- Plant Recommendation Engine: Suggest ideal plants based on climate data (OpenWeatherMap API), space (user input), and user preferences. Includes pros, cons, and care level for each plant.
- Plant Identification: Identify plant species from user-uploaded images using the Pl@ntNet API.
- Interactive Map: Display local green spaces, recommended planting areas, and nearby plant shops using Leaflet and Mapbox API. Shows medium inspired vlogs.
- Personalized Suggestions: Uses air quality data (AirVisual API) and soil moisture levels (NASA Soil Moisture API) to suggest air-purifying or water-efficient plants.
- Smart Suggestions tool: AI powered service that uses available environmental data and user provided plant information to determine whether to show the plant as a potential match.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke nature and growth.
- Background color: Light sage (#F1F8E9), a desaturated tint of forest green, to provide a calm backdrop.
- Accent color: Earthy brown (#795548), analogous to green, for highlights and calls to action.
- Headline font: 'Playfair' serif for elegant headings; body font: 'PT Sans' sans-serif for readability.
- Use a set of simple, outlined icons representing different plant types, environmental factors, and user actions. All icons should be in green.
- Minimalist layout with a focus on white space and clear content hierarchy. Use soothing images.
- Subtle transitions and animations to create a smooth and engaging user experience when switching pages or loading new data.
